Курсовая работа "Дюрация портфеля"

New

Основной целью этого проекта являлась разработка полноценного клиент-серверного приложения под названием "Дюрация", созданного с использованием языка программирования Java. Это приложение включает в себя серверную часть, построенную на фреймворке Spring Boot и обеспечивающую реализацию RESTful API для взаимодействия с клиентской частью. Клиентская часть представлена графическим интерфейсом JavaFX и способна взаимодействовать с сервером при помощи запросов к API Сервера.

Обе составляющие приложения – как серверная, так и клиентская части – разработаны в соответствии с моделью MVC, позволяющей эффективно разделять управляющую логику программы на отдельные компоненты. Использование Java и JVM (Java Virtual Machine) делает разработанное решение кроссплатформенным, что означает его способность работать на различных операционных системах без необходимости модификаций, обеспечивая универсальность и доступность для пользователей с разных платформ.
